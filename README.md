# Noxiss Store — Telegram Mini App

Готовый магазин цифровых товаров: Telegram-авторизация, баланс в USDT, пополнение через Crypto Pay, автоматическая/ручная выдача и админ-режим.

## Запуск
1. Создайте Neon PostgreSQL и выполните `db/schema.sql` в SQL Editor.
2. Скопируйте `.env.example` в `.env.local` и заполните значения **новыми** токенами.
3. Выполните `npm install && npm run dev`.
4. Импортируйте репозиторий в Vercel и добавьте те же Environment Variables.
5. В Crypto Pay задайте webhook: `https://ВАШ-АДРЕС/api/cryptopay/webhook`.
6. После деплоя выполните `npm run setup:bot` локально с переменными окружения.

## Управление выдачей
Администратор открывает вкладку «Админ» и выбирает для каждого товара:
- `Автоматически` — содержимое `delivery_content` выдаётся сразу;
- `Вручную` — создаётся заказ со статусом pending;
- `Приостановить` — кнопка покупки блокируется.

Тексты выдачи и товары хранятся в таблице `products`. API уже поддерживает их изменение. Не коммитьте `.env.local` и никогда не вставляйте токены в клиентский код.
